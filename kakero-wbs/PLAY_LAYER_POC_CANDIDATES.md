# Play Layer PoC Candidates

作成日: 2026-06-16

## 前提

KAKEROの本質的payoffは、人生が盤面になること。

ただし、入口と継続エンジンは遊び。  
遊びは単なる飾りではなく、ドパガキがしがみつくレベルの継続エンジンとして検証する。

卵は確定仕様ではない。  
卵は「突発イベント / 隠し報酬 / 無料ユーザー向け試食体験」の代表案として扱う。

## PoC候補

### 1. 突発イベント / 隠し報酬

| 項目 | 内容 |
| --- | --- |
| 何を検証するか | 予期しない報酬が、毎日開く理由になるか |
| どの画面で起きるか | トップ画面、人生コンパス、けんこうマップ閲覧後 |
| 想定するユーザー行動 | 盤面を見る、カードを直す、けんこうマップを開く、翌日も様子を見に来る |
| Goodhartリスク | ログイン連打、カード乱造、ポイント盛りで報酬を掘るリスク |
| 回避方針 | ログイン回数や数値指標に機械連動しない。有効利用の節目やランダム風演出に留める |
| walking skeleton | PoC候補として入れる。卵の形に固定しない |

### 2. 変な称号

| 項目 | 内容 |
| --- | --- |
| 何を検証するか | 変な称号が、盤面を見返す楽しさや共有したくなる感覚を生むか |
| どの画面で起きるか | トップ画面、プロフィール/状態表示、カード編集後 |
| 想定するユーザー行動 | 自分の状態を見返す、称号の変化を見に戻る、盤面への愛着が増える |
| Goodhartリスク | 称号獲得のためにカードや状態を嘘にするリスク |
| 回避方針 | 称号は進捗量ではなく、状態の解釈や節目に出す。良い状態だけを報酬化しない |
| walking skeleton | 軽量PoCとして入れる候補 |

### 3. きこうか君育成 / 変化

| 項目 | 内容 |
| --- | --- |
| 何を検証するか | きこうか君の見た目・口調・反応の小さな変化が、相棒感と継続理由になるか |
| どの画面で起きるか | チュートリアル、願望入力、今日タスク、原則確認ページ |
| 想定するユーザー行動 | きこうか君に話す、カードを追加する、盤面を整える、翌日も会いに来る |
| Goodhartリスク | なつき度や成長値を上げるために無意味な入力を増やすリスク |
| 回避方針 | 数値ゲージを前面に出さず、盤面理解や節目に応じた表情/文言変化に留める |
| walking skeleton | 後回し寄り。ただしチュートリアルの表情づけとして小さく入れる余地あり |

## 今回の結論

walking skeletonで優先するPoCは以下。

1. 突発イベント / 隠し報酬
2. 変な称号

きこうか君育成 / 変化は有望だが、実装が膨らみやすいため、まずはチュートリアルや原則確認ページの表情づけに留める。
